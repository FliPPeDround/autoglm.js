# miGPT

智能音箱与手机自动化的融合助手，通过小爱音箱接收指令，智能判断并执行手机操作任务。

## 功能特性

- 🎤 **语音交互**：通过小爱音箱接收用户语音指令
- 🧠 **智能判断**：使用 GLM-4 模型分析任务，自动判断是否需要手机操作
- 📱 **手机控制**：集成 AutoGLM 实现手机应用自动化操作
- 🏠 **智能家居**：支持小爱音箱原生智能家居控制功能

## 技术栈

- **AI 模型**：GLM-4 (通过 LangChain)
- **音箱控制**：@mi-gpt/next
- **手机控制**：autoglm.js

## 安装

```bash
pnpm install
```

## 运行

```bash
pnpm start
```

## 使用说明

1. 配置小爱音箱账号信息（在 `.env` 文件中）
2. 配置 AutoGLM API 密钥（在 `.env` 文件中）
3. 启动服务后，可通过以下方式触发手机操作：
   - 直接说关键词："用手机"、"手机"、"帮我用手机"
   - 或者系统会自动判断任务是否需要手机操作，有因为是大模型自主判断，所以可能会有误判且有一定延迟

## 示例

"用手机打开淘宝闪购，点一杯霸王茶姬的伯牙绝弦我要大杯热的"
